# Course Prerequisites Prep

## Assumptions

We will assume that Solidity is not your first programming language. No prior knowledge of Ethereum is required. However, understanding the basics of blockchains and Ethereum will help.

As stated before, we will assume:

- At least six months to one year of programming experience.
- Familiarity with JavaScript.
- Basic experience with the command line.
- Basic experience with NodeJS.
- Basic experience with React.
- Basic experience with Git and Github.

## How do I prepare to become a Solidity Developer

1. Bookmark this website to return to it.
2. Engage with the resources below. Start with the ConsenSys Academy Basic training course and supplement your skills with Codecademy and Freecodecamp.
3. Join a community. Asking question will help save you time and headaches.

## Prerequisite Resources

For those without programming experience, you can check out the following resources:

- [ConsenSys Academy Basic Training](https://courses.consensys.net/courses/bootcamp-basic-training)
- [FreeCodeCamp.org](https://www.freecodecamp.org/)
- [Codecademy](https://www.codecademy.com/)
- [The Coding Train](https://thecodingtrain.com/) <3

Use the ConsenSys Academy Basic Training as a guide to develop the prerequisite skills.
